# [Deep BSDE Solver](https://doi.org/10.1073/pnas.1718942115) in PyTorch

## Quick Installation

```bash
python3.11 -m venv .venv
python --version

# Python 3.11.9
```

For a quick installation, you can create a Python virtual environment and install the required packages using pip:
```bash
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate

pip install -r requirements.txt
```
